## Theory

For end users, installation should be one simple command.  
Global install makes the `monalisa` command available in your terminal.  
After install, you can run `.priyo` files from any folder.  
This is the official user-facing install method right now.

<Callout title="Current install path" type="idea">
  Use npm global install. Standalone installer/binary flow is not the official stable user path yet.
</Callout>

## Prerequisite

- Node.js 18+ with npm

## Install PriyoScript globally

```bash
npm i -g priyoscript
```

## Verify installation

```bash
monalisa -h
monalisa -repl
```

## Run your first file

Create `hello.priyo`:

```priyo
monalisa {
  priyoTell("Hello, PriyoScript!")
}
```

Run:

```bash
monalisa hello.priyo
```

Expected output:

```text
Hello, PriyoScript!
```

## Update PriyoScript

```bash
npm i -g priyoscript@latest
```

## Uninstall PriyoScript

```bash
npm uninstall -g priyoscript
```

<Callout title="Troubleshooting" type="error">
  If `monalisa` is not recognized, restart terminal and ensure your npm global bin path is in PATH.
</Callout>
