## Theory

Installation means preparing two things:

- The PriyoScript executable/CLI
- The environment path so terminal can find it

## Option A: Run from source

```bash
npm install
node bin/monalisa.js -h
```

## Option B: Windows executable

```bash
npm run dist:win
```

Then:

```powershell
.\dist\priyoscript.exe -h
```

## How execution works (very simple)

Think of `monalisa` like a remote control button:

1. You press command in terminal
2. CLI opens your `.priyo` file
3. PriyoScript pipeline runs (read -> parse -> compile -> execute)
4. Output appears in terminal

## Verify

```bash
monalisa -h
monalisa -repl
```

<Callout title="Troubleshooting" type="error">
  If `monalisa` is not recognized, restart terminal and check PATH.
</Callout>
