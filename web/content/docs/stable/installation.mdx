# Installation

## Prerequisites

- Node.js 18+
- npm

## Run from Source (recommended for contributors)

```bash title="from repository root"
npm install
node bin/monalisa.js -h
```

<Callout title="Tip" type="idea">
  Use source mode while developing language/compiler/runtime features.
</Callout>

## Build Windows executable

```bash
npm run dist:win
```

Then run:

```powershell
.\dist\priyoscript.exe -h
```

## Optional user install for PATH

```powershell
powershell -ExecutionPolicy Bypass -File scripts/install-windows.ps1
```

## Verify

```bash title="check CLI and REPL"
monalisa -h
monalisa -repl
```

<Callout title="Troubleshooting" type="error">
  If `monalisa` is not found, restart terminal and verify your PATH includes the install folder.
</Callout>
