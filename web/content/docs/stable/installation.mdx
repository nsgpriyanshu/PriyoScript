## Installation models

PriyoScript supports two common workflows:

- Source workflow (best for contributors)
- Windows binary workflow (best for end users)

## Prerequisites

- Node.js 18+
- npm

## Option A: Run from source

```bash title="from repository root"
npm install
node bin/monalisa.js -h
```

You can run files directly:

```bash
node bin/monalisa.js examples/main.priyo
```

## Option B: Build Windows executable

```bash
npm run dist:win
```

Then verify:

```powershell
.\dist\priyoscript.exe -h
```

Optional PATH install for current user:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/install-windows.ps1
```

## Verify CLI + REPL

```bash
monalisa -h
monalisa -repl
```

<Callout title="Troubleshooting" type="error">
  If `monalisa` is not recognized, restart terminal and verify PATH includes the install directory.
</Callout>

## Which option should you choose?

- Choose source mode if you are editing language/runtime code
- Choose exe mode if you only need to run `.priyo` programs
