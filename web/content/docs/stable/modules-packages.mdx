# Modules & Packages

## Built-in package import

```priyo
monalisa {
  lisaaBring math
  priyoTell(math.cube(2))
}
```

## User module export

`college.priyo`

```priyo
lisaaBox {
  lisaaTask square(x) {
    priyoGiveBack x * x
  }

  priyoKeep campus = "NSEC"

  lisaaShare square
  lisaaShare campus
}
```

`main.priyo`

```priyo
monalisa {
  lisaaBring "./college.priyo": college

  priyoTell(college.campus)
  priyoTell(college.square(5))
}
```

## Named and aliased imports

```priyo
monalisa {
  lisaaBring "./college.priyo": [campus, square: squareFn]
  priyoTell(campus)
  priyoTell(squareFn(6))
}
```

<Callout title="Module rule" type="warn">
  Path imports must target files that expose a `lisaaBox { ... }` module body.
</Callout>
