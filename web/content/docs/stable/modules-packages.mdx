# Modules & Packages

## Built-in package import

```js
lisaaBring math
priyoTell(math.cube(2))
```

## User module

`college.priyo`

```js
lisaaBox {
  lisaaTask square(x) {
    priyoGiveBack x * x
  }

  priyoKeep campus = "NSEC"
  lisaaShare square
  lisaaShare campus
}
```

`main.priyo`

```js
monalisa {
  lisaaBring "./college.priyo": college
  priyoTell(college.campus)
  priyoTell(college.square(5))
}
```

## Named and aliased imports

```js
lisaaBring "./college.priyo": [campus, cgpaToGradePoint: gradeFn]
lisaaBring "./college.priyo": collegeModule
```

<Callout title="Title" type="warn">
Path imports must point to modules that start with `lisaaBox { ... }`.
</Callout>
