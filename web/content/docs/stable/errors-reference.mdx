---
title: Error Codes Reference
description: Stable reference for PriyoScript syntax, compile, runtime, and engine error codes.
---

Use with CLI helpers:

```bash
monalisa -errors
monalisa -explain <ERROR_CODE>
```

<Callout title="How to use this page" type="idea">
  Find the code from CLI/REPL output, then open the matching section below.
</Callout>

## Syntax stage

| Code       | Meaning                                |
| ---------- | -------------------------------------- |
| `PSYN-001` | Generic syntax parsing failure         |
| `PSYN-002` | Unexpected token/keyword or typo       |
| `PSYN-003` | Reserved/blocked word used incorrectly |

## Compile stage

| Code       | Meaning                                   |
| ---------- | ----------------------------------------- |
| `PCMP-001` | Compile pipeline failed after parse stage |

## Runtime stage

| Code       | Meaning                                         |
| ---------- | ----------------------------------------------- |
| `PRUN-000` | Generic runtime failure                         |
| `PRUN-101` | Variable used before declaration / out of scope |
| `PRUN-102` | Reassignment of `priyoPromise`                  |
| `PRUN-103` | Division or modulo by zero                      |
| `PRUN-104` | Unknown class reference                         |
| `PRUN-105` | Unknown function/method reference               |
| `PRUN-106` | Invalid/missing property access                 |
| `PRUN-107` | Invalid call argument count/type                |
| `PRUN-108` | Input type mismatch                             |
| `PRUN-109` | Unknown built-in package                        |
| `PRUN-110` | Module import path failure                      |

## Engine stage

| Code       | Meaning                        |
| ---------- | ------------------------------ |
| `PENG-001` | Internal engine/compiler fault |

<Callout title="Need details?" type="warn">
  Run `monalisa -explain <ERROR_CODE>` to get longer CLI help for the same code.
</Callout>
