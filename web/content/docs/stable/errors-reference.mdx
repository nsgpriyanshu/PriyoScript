## Theory

Error codes make debugging fast because each code maps to one problem family.

## How to use

```bash
monalisa -errors
monalisa -explain <ERROR_CODE>
```

## `PSYN-002` Unexpected token/keyword

Why:
- Keyword typo or invalid token order.

Wrong:

```priyo
monalisa {
  prakritiIff (priyoTrue) {
    priyoTell("ok")
  }
}
```

Correct:

```priyo
monalisa {
  prakritiIf (priyoTrue) {
    priyoTell("ok")
  }
}
```

## `PRUN-101` Variable scope error

Why:
- Variable used before declaration or outside scope.

Wrong:

```priyo
monalisa {
  priyoTell(total)
  priyoKeep total = 10
}
```

Correct:

```priyo
monalisa {
  priyoKeep total = 10
  priyoTell(total)
}
```

## `PRUN-102` Cannot reassign `priyoPromise`

Wrong:

```priyo
monalisa {
  priyoPromise x = 1
  x = 2
}
```

Correct:

```priyo
monalisa {
  priyoChange x = 1
  x = 2
}
```

## `PRUN-103` Division by zero

Wrong:

```priyo
monalisa {
  priyoTell(10 / 0)
}
```

Correct:

```priyo
monalisa {
  lisaaTask safeDivide(a, b) {
    prakritiIf (b == 0) {
      priyoGiveBack priyoEmpty
    }
    priyoGiveBack a / b
  }
}
```

## `PRUN-109` Unknown package

Wrong:

```priyo
monalisa {
  lisaaBring maths
}
```

Correct:

```priyo
monalisa {
  lisaaBring math
}
```

## `PRUN-110` Module path failed

Wrong:

```priyo
monalisa {
  lisaaBring "./college.priyoo": college
}
```

Correct:

```priyo
monalisa {
  lisaaBring "./college.priyo": college
}
```

## How execution works (simple)

When error occurs:

1. Engine adds error code
2. Adds where it happened (line/column)
3. Adds tip and docs link
4. Stops unsafe execution
