## Theory

Functions are reusable instruction groups. They reduce duplication and make code easier to test.

## Example 1: Basic function

```priyo
monalisa {
  lisaaTask add(a, b) {
    priyoGiveBack a + b
  }
  priyoTell(add(10, 5))
}
```

## Example 2: Recursion

```priyo
monalisa {
  lisaaTask factorial(n) {
    prakritiIf (n <= 1) {
      priyoGiveBack 1
    }
    priyoGiveBack n * factorial(n - 1)
  }
  priyoTell(factorial(5))
}
```

## Example 3: Closure

```priyo
monalisa {
  lisaaTask makeCounter() {
    priyoChange count = 0
    lisaaTask next() {
      count = count + 1
      priyoGiveBack count
    }
    priyoGiveBack next
  }
  priyoKeep c = makeCounter()
  priyoTell(c())
}
```

## Example 4: Business rule helper

```priyo
monalisa {
  lisaaTask isAdult(age) {
    priyoGiveBack age >= 18
  }
  priyoTell(isAdult(21))
}
```

## How execution works (simple)

Think of function like a toy box:

1. Put steps in box (`lisaaTask`)
2. Call box by name
3. Box runs steps
4. Gives result back (`priyoGiveBack`)

<Callout title="Rule" type="warn">
  `priyoGiveBack` can be used only inside function blocks.
</Callout>
