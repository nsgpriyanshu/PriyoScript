## Theory

Control flow is how your program decides:

- Which block to run (`if`, `switch`)
- How long to run (`while`, `for`)
- When to stop (`break`, `continue`)

## Example 1: If / else-if / else

```priyo
monalisa {
  priyoKeep score = 82
  prakritiIf (score >= 90) {
    priyoTell("A")
  } prakritiElseIf (score >= 80) {
    priyoTell("B")
  } prakritiElse {
    priyoTell("C")
  }
}
```

## Example 2: While loop

```priyo
monalisa {
  priyoKeep i = 0
  prakritiAsLongAs (i < 3) {
    priyoTell(i)
    i = i + 1
  }
}
```

## Example 3: For loop

```priyo
monalisa {
  prakritiCount (priyoChange i = 1; i <= 5; i = i + 1) {
    priyoTell(i)
  }
}
```

## Example 4: For-each + switch

```priyo
monalisa {
  priyoKeep tags = ["news", "music"]
  prakritiCount (tag priyoInside tags) {
    prakritiChoose (tag) {
      prakritiCase ("music") { priyoTell("Audio") }
      prakritiOtherwise { priyoTell("General") }
    }
  }
}
```

## How execution works (like a child)

Imagine traffic lights:

- Green (`if true`) -> go this way
- Red (`else`) -> go other way
- Round track (`loop`) -> keep going until condition says stop

Language mapping:

- JS/Java/C: similar `if/for/while/switch`
- Python: `elif` and `for in range(...)`
