# Quick Start

Create `hello.priyo`:

```js title="hello.priyo"
monalisa {
  priyoTell("Hello from PriyoScript")
}
```

Run it:

```bash
monalisa hello.priyo
```

## Input + output example

```js title="input.priyo"
monalisa {
  priyoKeep name = priyoListenSentence("Your name: ")
  priyoKeep age = priyoListenNumber("Your age: ")
  priyoTell.Success("Hi " + name + ", age " + age)
}
```

## Variables and expressions

```js title="variables.priyo"
monalisa {
  priyoKeep x = 10
  priyoChange y = 5
  priyoPromise z = x * y
  priyoTell(z)
}
```

<Callout title="Title" type="warn">
  Use `priyoPromise` only when value should never be reassigned.
</Callout>
