## Theory

A program is just instructions executed top to bottom unless control-flow changes the order.

## Example 1: Hello World

```priyo
monalisa {
  priyoTell("Hello from PriyoScript")
}
```

## Example 2: Input + output

```priyo
monalisa {
  priyoKeep name = priyoListenSentence("Your name: ")
  priyoKeep age = priyoListenNumber("Your age: ")
  priyoTell.Success("Hi " + name)
  priyoTell.Info("Age: " + age)
}
```

## Example 3: Variables + math

```priyo
monalisa {
  priyoKeep base = 12
  priyoChange tax = 3
  priyoPromise total = base + tax
  priyoTell(total)
}
```

## Example 4: Small function

```priyo
monalisa {
  lisaaTask areaSquare(side) {
    priyoGiveBack side * side
  }
  priyoTell(areaSquare(5))
}
```

## How execution works (like a 5-year-old)

Imagine robot steps:

1. Robot enters `monalisa { ... }`
2. Robot reads line 1, does it
3. Reads next line, does it
4. When robot sees function call, it jumps into function and comes back with answer

Equivalent hello world:

```js
console.log('Hello from JavaScript')
```

```python
print('Hello from Python')
```
