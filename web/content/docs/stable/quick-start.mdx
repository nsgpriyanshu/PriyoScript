## Goal

This page helps you move from installation to working PriyoScript code quickly.

By the end, you will be able to:

- Run a `.priyo` file with `monalisa`
- Read input and print output
- Use variables and functions

## Step 1: Hello World

Create `hello.priyo`:

```priyo
monalisa {
  priyoTell("Hello from PriyoScript")
}
```

Run:

```bash
monalisa hello.priyo
```

Expected output:

```text
Hello from PriyoScript
```

## Step 2: Input and output

```priyo
monalisa {
  priyoKeep name = priyoListenSentence("Your name: ")
  priyoKeep age = priyoListenNumber("Your age: ")

  priyoTell.Success("Hi " + name)
  priyoTell.Info("Age: " + age)
}
```

## Step 3: Variables and expressions

```priyo
monalisa {
  priyoKeep base = 12
  priyoChange tax = 3
  priyoPromise total = base + tax

  priyoTell(total)
}
```

## Step 4: First function

```priyo
monalisa {
  lisaaTask areaSquare(side) {
    priyoGiveBack side * side
  }

  priyoTell(areaSquare(5))
}
```

Expected output:

```text
25
```

## Execution flow

1. Execution starts inside `monalisa { ... }`.
2. Statements run in sequence from top to bottom.
3. Variable declarations store values in runtime memory.
4. Function definitions are registered.
5. Function calls evaluate arguments, execute function body, and return results.
6. `priyoTell` prints final values.

## Quick comparison

JavaScript:

```js
console.log('Hello from JavaScript')
```

Python:

```python

```

PriyoScript:

