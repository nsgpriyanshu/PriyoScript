## Goal

By the end of this page, you will be able to:

- Write and run a `.priyo` file
- Read user input and print output
- Use variables and expressions correctly

## Example 1: Hello World

```priyo title="hello.priyo"
monalisa {
  priyoTell("Hello from PriyoScript")
}
```

```bash
monalisa hello.priyo
```

Equivalent (JavaScript):

```js
console.log('Hello from JavaScript')
```

Equivalent (Python):

```python
print("Hello from Python")
```

## Example 2: Input + output

```priyo title="input.priyo"
monalisa {
  priyoKeep name = priyoListenSentence("Your name: ")
  priyoKeep age = priyoListenNumber("Your age: ")

  priyoTell.Success("Hi " + name)
  priyoTell.Info("Age: " + age)
}
```

## Example 3: Variables + math

```priyo title="variables.priyo"
monalisa {
  priyoKeep base = 12
  priyoChange tax = 3
  priyoPromise total = base + tax

  priyoTell(total)
}
```

## Example 4: Small function

```priyo title="function.priyo"
monalisa {
  lisaaTask areaSquare(side) {
    priyoGiveBack side * side
  }

  priyoTell(areaSquare(5))
}
```

<Callout title="Variable rule" type="idea">
  Use `priyoPromise` only when you do not want reassignment.
</Callout>
