# Quick Start

This page gets you from zero to a working PriyoScript program in a few minutes.

## 1. Hello World

Create `hello.priyo`:

```priyo title="hello.priyo"
monalisa {
  priyoTell("Hello from PriyoScript")
}
```

Run it:

```bash
monalisa hello.priyo
```

## 2. Read input and print output

```priyo title="input.priyo"
monalisa {
  priyoKeep name = priyoListenSentence("Your name: ")
  priyoKeep age = priyoListenNumber("Your age: ")

  priyoTell.Success("Hi " + name)
  priyoTell.Info("Age: " + age)
}
```

## 3. Variables and expressions

```priyo title="variables.priyo"
monalisa {
  priyoKeep x = 10
  priyoChange y = 5
  priyoPromise result = x * y

  priyoTell(result)
}
```

<Callout title="When to use each variable type" type="idea">
  Use `priyoKeep` for function-scoped mutable values, `priyoChange` for block-scoped mutable values,
  and `priyoPromise` for values that must not be reassigned.
</Callout>
