# Classes & OOP

## Class and constructor

```priyo
monalisa {
  lisaaFamily Student {
    lisaaTask init(name) {
      priyoSelf.name = name
    }

    lisaaTask intro() {
      priyoTell("Hi, I am " + priyoSelf.name)
    }
  }

  priyoKeep s = priyoCreate Student("Priyo")
  s.intro()
}
```

## Inheritance

```priyo
monalisa {
  lisaaFamily Person {
    lisaaTask init(name) {
      priyoSelf.name = name
    }
  }

  lisaaFamily Teacher lisaaInherit Person {
    lisaaTask init(name, subject) {
      priyoParent(name)
      priyoSelf.subject = subject
    }
  }
}
```

`priyoParent(...)` inside child `init` must be called first, and called only once.

## Static fields and methods

```priyo
monalisa {
  lisaaFamily MathUtil {
    lisaaStable priyoPromise PI = 3.14159

    lisaaStable lisaaTask square(n) {
      priyoGiveBack n * n
    }
  }

  priyoTell(MathUtil.square(4))
}
```

<Callout title="Constructor safety" type="error">
  If a child constructor violates `priyoParent(...)` rules, PriyoScript raises a compile-time error.
</Callout>
