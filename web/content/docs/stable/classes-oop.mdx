# Classes & OOP

## Class and Constructor

```js
lisaaFamily Student {
  lisaaTask init(name) {
    priyoSelf.name = name
  }
}
```

## Create Instance

```js
priyoKeep s = priyoCreate Student("Priyo")
```

## Inheritance

```js
lisaaFamily Person {
  lisaaTask init(name) {
    priyoSelf.name = name
  }
}

lisaaFamily Teacher lisaaInherit Person {
  lisaaTask init(name, subject) {
    priyoParent(name)
    priyoSelf.subject = subject
  }
}
```

`priyoParent(...)` in child `init` must be first and only once.

## Static methods and fields

```js
lisaaFamily MathUtil {
  lisaaStable priyoPromise PI = 3.14159

  lisaaStable lisaaTask square(n) {
    priyoGiveBack n * n
  }
}

priyoTell(MathUtil.square(4))
```

<Callout title="Title" type="error">
  If child `init` does not call `priyoParent(...)` as first statement, compilation fails.
</Callout>
