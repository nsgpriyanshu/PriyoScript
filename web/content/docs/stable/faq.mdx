# FAQ

## Is PriyoScript production-ready?

Core language/runtime features are stable, but PriyoScript is still evolving.

## Does it support modules?

Yes. Use `lisaaBring` with built-in packages and local module paths.

## Is there a REPL?

Yes. Run `monalisa -repl`.

## Where do I find error explanations?

Use:

```bash
monalisa -errors
monalisa -explain <ERROR_CODE>
```

## Why are PriyoScript code blocks highlighted as JS here?

For now, docs use JavaScript lexer mode to provide colorized reading.
A dedicated PriyoScript syntax-highlighting grammar can be added later.

<Callout title="Title" type="warn">
  Syntax highlighting style does not change actual PriyoScript semantics.
</Callout>
