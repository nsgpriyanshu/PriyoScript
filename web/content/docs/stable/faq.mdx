## Is PriyoScript production-ready?

Core language/runtime features are stable, but the project is still actively evolving.

## Does PriyoScript support modules?

Yes. Use `lisaaBring` with built-in packages and file-based modules.

## Is there an interactive shell?

Yes. Run `monalisa -repl`.

## How do I read error details quickly?

Use:

```bash
monalisa -errors
monalisa -explain <ERROR_CODE>
```

## Is syntax highlighting in docs native PriyoScript?

Yes. Stable docs use PriyoScript-aware highlighting in code fences.

## Where should beginners start?

Start with [Quick Start](./quick-start), then [Syntax Basics](./syntax-basics), then [Control Flow](./control-flow).

<Callout title="Learning order" type="idea">
  Do not jump directly to modules/OOP if fundamentals are unclear. PriyoScript is easiest when
  learned in sequence.
</Callout>
