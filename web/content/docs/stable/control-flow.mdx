## Theory

Control flow decides which statements run, how many times, and under what conditions.

## Example 1: If / else-if / else

```priyo
monalisa {
  priyoKeep score = 82

  prakritiIf (score >= 90) {
    priyoTell("A")
  } prakritiElseIf (score >= 80) {
    priyoTell("B")
  } prakritiElse {
    priyoTell("C")
  }
}
```

Equivalent (JavaScript): `if / else if / else`
Equivalent (Python): `if / elif / else`

## Example 2: While loop

```priyo
monalisa {
  priyoKeep i = 0
  prakritiAsLongAs (i < 3) {
    priyoTell(i)
    i = i + 1
  }
}
```

## Example 3: Counted loop

```priyo
monalisa {
  prakritiCount (priyoChange i = 1; i <= 5; i = i + 1) {
    priyoTell(i)
  }
}
```

Comparable forms:

- C / Java / JavaScript: `for (init; cond; step)`
- Python: usually `for i in range(...)`

## Example 4: For-each + switch

```priyo
monalisa {
  priyoKeep tags = ["news", "sports", "music"]

  prakritiCount (tag priyoInside tags) {
    prakritiChoose (tag) {
      prakritiCase ("music") { priyoTell("Audio") }
      prakritiOtherwise { priyoTell("General") }
    }
  }
}
```

<Callout title="Loop control" type="idea">
  Use `prakritiStop` to break and `prakritiGoOn` to skip the current loop iteration.
</Callout>
