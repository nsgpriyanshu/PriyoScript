# Control Flow

## If / Else

```txt
prakritiIf (score > 80) {
  priyoTell("Great")
} prakritiElse {
  priyoTell("Keep going")
}
```

## While

```txt
priyoKeep i = 0
prakritiAsLongAs (i < 3) {
  priyoTell(i)
  i = i + 1
}
```

## For

```txt
prakritiCount (priyoChange i = 0; i < 5; i = i + 1) {
  priyoTell(i)
}
```

## Switch

```txt
prakritiChoose (role) {
  prakritiCase ("admin") { priyoTell("Full access") }
  prakritiOtherwise { priyoTell("Limited access") }
}
```

## Break / Continue

- `prakritiStop` breaks loop or switch
- `prakritiGoOn` continues current loop
