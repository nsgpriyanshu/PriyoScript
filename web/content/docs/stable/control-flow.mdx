# Control Flow

## Conditional branches

```priyo
monalisa {
  priyoKeep score = 85

  prakritiIf (score >= 90) {
    priyoTell.Success("Excellent")
  } prakritiElseIf (score >= 75) {
    priyoTell.Info("Good")
  } prakritiElse {
    priyoTell("Keep practicing")
  }
}
```

## While loop

```priyo
monalisa {
  priyoKeep i = 0

  prakritiAsLongAs (i < 3) {
    priyoTell(i)
    i = i + 1
  }
}
```

## Counted loop

```priyo
monalisa {
  prakritiCount (priyoChange i = 0; i < 5; i = i + 1) {
    priyoTell(i)
  }
}
```

## For-each over arrays

```priyo
monalisa {
  priyoKeep items = ["A", "B", "C"]

  prakritiCount (item priyoInside items) {
    priyoTell(item)
  }
}
```

## Switch style branching

```priyo
monalisa {
  priyoKeep role = "admin"

  prakritiChoose (role) {
    prakritiCase ("admin") { priyoTell("Full access") }
    prakritiCase ("editor") { priyoTell("Edit access") }
    prakritiOtherwise { priyoTell("Read only") }
  }
}
```

## Loop control

- `prakritiStop` exits current loop or switch
- `prakritiGoOn` skips current iteration and continues loop
