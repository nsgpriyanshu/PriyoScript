## CLI essentials

```bash
monalisa -h
monalisa -syntax
monalisa -errors
monalisa -explain PSYN-002
monalisa file.priyo
monalisa -repl
```

## REPL command set

- `.help` show command list
- `.clear` clear multiline buffer
- `.reset` reset runtime state and module cache
- `.load <file.priyo>` run file in current REPL scope
- `.exit` quit REPL

## Example session

```text
priyo> priyoKeep x = 12
priyo> priyoTell(x * 2)
24
priyo> .help
```

## Suggested workflow

1. Prototype expressions in REPL
2. Move stable logic into `.priyo` files
3. Run from CLI for repeatable behavior
4. Use error codes + docs when failures occur

<Callout title="Pro tip" type="idea">
  Keep REPL for experiments and CLI files for source-controlled project code.
</Callout>
