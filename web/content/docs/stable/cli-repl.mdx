## Theory

CLI is best for running files. REPL is best for quick experiments.

## Common CLI commands

```bash
monalisa -h
monalisa -syntax
monalisa -errors
monalisa -explain PSYN-002
monalisa file.priyo
monalisa -repl
```

## REPL commands

- `.help`
- `.clear`
- `.reset`
- `.load <file.priyo>`
- `.exit`

## Example REPL session

```text
priyo> priyoKeep x = 12
priyo> priyoTell(x * 2)
24
```

## How execution works (simple)

- CLI mode: run full file once
- REPL mode: run one snippet at a time, keep memory between inputs

<Callout title="Workflow tip" type="idea">
  Prototype in REPL, then move stable code to `.priyo` files.
</Callout>
