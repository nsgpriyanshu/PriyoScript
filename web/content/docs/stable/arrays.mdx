## 1. Theory

An array stores multiple values in one variable.  
Each value has a numeric position called index.  
Use arrays when you have a list of related items.  
PriyoScript arrays are first-class values.  
You can read, update, slice, loop, destructure, and transform arrays.  
Helpers like `map`, `filter`, and `reduce` make collection work clean.  
Learning arrays is essential for real applications and data handling.  
Arrays also build the base for matrix-style programs.

## 2. Program Examples

### Example 1: Indexing and Update

```priyo
monalisa {
  priyoKeep nums = [10, 20, 30]
  priyoTell(nums[0])
  nums[1] = 25
  priyoTell(nums[1])
}
```

Output:

```text
10
25
```

### Example 2: Slicing

```priyo
monalisa {
  priyoKeep nums = [10, 20, 30, 40]
  priyoTell(nums[0:2])
  priyoTell(nums[:2])
  priyoTell(nums[1:])
}
```

Output:

```text
[10, 20]
[10, 20]
[20, 30, 40]
```

### Example 3: Destructuring With Default

```priyo
monalisa {
  priyoKeep values = [7, 8]
  priyoChange[(first, second, third = 99)] = values
  priyoTell(first)
  priyoTell(second)
  priyoTell(third)
}
```

Output:

```text
7
8
99
```

### Example 5: map / filter / reduce

```priyo
monalisa {
  lisaaTask isEven(x) {
    priyoGiveBack x % 2 == 0
  }

  lisaaTask add(acc, x) {
    priyoGiveBack acc + x
  }

  priyoKeep values = [1, 2, 3, 4, 5]
  priyoKeep evens = priyoArray.filter(values, isEven)
  priyoKeep sum = priyoArray.reduce(evens, add, 0)

  priyoTell(evens)
  priyoTell(sum)
}
```

Output:

```text
[2, 4]
6
```

## 3. Execution Flow

1. Array literal allocates ordered storage.
2. `arr[i]` reads value at index `i`.
3. `arr[i] = value` updates one slot.
4. Slice builds a new array from range.
5. For-each loop reads each item one by one.
6. `filter` keeps matching items.
7. `reduce` combines array into one final value.

## 4. Comparison With Other Languages

C arrays:

```c
int nums[3] = {10, 20, 30};
printf("%d\n", nums[0]);
```

Java arrays:

```java
int[] nums = {10, 20, 30};
System.out.println(nums[0]);
```

JavaScript arrays:

```javascript
const nums = [10, 20, 30]
console.log(nums[0])
```

Python lists:

```python
nums = [10, 20, 30]
print(nums[0])
```

PriyoScript:

```priyo
monalisa {
  priyoKeep nums = [10, 20, 30]
  priyoTell(nums[0])
}
```

Difference summary:

- C arrays are low-level and fixed-size.
- Java arrays are typed and fixed-size.
- JavaScript/Python/PriyoScript arrays are dynamic and ergonomic.
- PriyoScript adds readable helper APIs through `priyoArray`.
