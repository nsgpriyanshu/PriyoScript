# Arrays

## Literals and Indexing

```txt
priyoKeep nums = [10, 20, 30]
priyoTell(nums[0])
nums[1] = 25
```

## Slicing

```txt
priyoTell(nums[0:2])
priyoTell(nums[:2])
priyoTell(nums[1:])
```

## Destructuring

```txt
priyoChange [first, second = 99] = nums
```

## Helpers

`priyoArray` provides helpers like:

- `length`, `push`, `pop`
- `slice`, `includes`, `indexOf`
- `map`, `filter`, `reduce`, `find`, `some`, `every`
