## Theory

An array stores many values in order. Each value has a position (index).

## Example 1: Create + read + update

```priyo
monalisa {
  priyoKeep nums = [10, 20, 30]
  priyoTell(nums[0])
  nums[1] = 25
  priyoTell(nums)
}
```

## Example 2: Slicing

```priyo
monalisa {
  priyoKeep nums = [10, 20, 30, 40]
  priyoTell(nums[0:2])
  priyoTell(nums[:2])
  priyoTell(nums[1:])
}
```

## Example 3: Destructuring

```priyo
monalisa {
  priyoKeep values = [7, 8]
  priyoChange[(first, second, third = 99)] = values
  priyoTell(third)
}
```

## Example 4: Helpers

```priyo
monalisa {
  lisaaTask isEven(x) {
    priyoGiveBack x % 2 == 0
  }
  priyoKeep values = [1, 2, 3, 4, 5]
  priyoKeep evens = priyoArray.filter(values, isEven)
  priyoTell(priyoArray.length(evens))
}
```

## How execution works (like a child)

Imagine toy boxes in a line:

- Box 0, box 1, box 2...
- `nums[1]` means open second box
- slice means take a group of nearby boxes
