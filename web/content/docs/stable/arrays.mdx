# Arrays

## Create and access

```priyo

  priyoKeep nums = [10, 20, 30]
  priyoTell(nums[0])

  priyoTell(nums)
```

## Slicing

```priyo

  priyoKeep nums = [10, 20, 30, 40]

  priyoTell(nums[0:2])
  priyoTell(nums[:2])
  priyoTell(nums[1:])
}
```

## Destructuring

```priyo

  priyoKeep values = [7, 8, 9]
  priyoChange[(first, second = 99, third)] = values

  priyoTell(first)

```

## Helper API (`priyoArray`)

- Query helpers: `length`, `includes`, `indexOf`
- Mutation helpers: `push`, `pop`, `slice`
- Callback helpers: `map`, `filter`, `reduce`, `find`, `some`, `every`

```priyo
monalisa {
  lisaaTask isEven(x) {
    priyoGiveBack x % 2 == 0
  }

  priyoKeep values = [1, 2, 3, 4, 5]
  priyoKeep evens = priyoArray.filter(values, isEven)
  priyoTell(priyoArray.length(evens))
}
```

<Callout title="Recommendation" type="idea">
  Prefer `priyoArray` helpers for transformations. They make intent clearer than manual loops.
</Callout>
