# Syntax Basics

## Entry Block

Every executable PriyoScript program starts with:

```js
monalisa {
  // statements
}
```

## Variables

```js
priyoKeep a = 10      // var-style, function-scoped
priyoChange b = 20    // let-style, block-scoped
priyoPromise c = 30   // const-style, block-scoped
```

<Callout title="Title" type="warn">
  Use `priyoPromise` only for values that must not be reassigned.
</Callout>

## Literals

- Number: `42`
- String: `"hello"`
- Boolean: `priyoTrue`, `priyoFalse`
- Null: `priyoNull`

## Expressions

```js
priyoKeep total = 5 + 10 * 2
priyoKeep ok = total >= 20 && priyoTrue
```

## Comments

```js
// single-line comment
/*
multi-line comment
*/
```

## Assignment targets

```js
monalisa {
  priyoKeep profile = priyoCreate Person()
  profile.name = "Priyo"
}
```
