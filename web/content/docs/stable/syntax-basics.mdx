# Syntax Basics

## Entry block

Every executable PriyoScript program starts with `monalisa { ... }`.

```priyo
monalisa {
  // statements
}
```

<Callout title="Required" type="warn">
  Top-level executable scripts must use one `monalisa` block.
</Callout>

## Variables

```priyo
priyoKeep a = 10
priyoChange b = 20
priyoPromise c = 30
```

- `priyoKeep`: function scope, mutable
- `priyoChange`: block scope, mutable
- `priyoPromise`: block scope, immutable reference

## Literals

- Number: `42`, `3.14`
- String: `"hello"`
- Boolean: `priyoTrue`, `priyoFalse`
- Empty value: `priyoEmpty`

## Expressions

```priyo
monalisa {
  priyoKeep total = 5 + 10 * 2
  priyoKeep ok = total >= 20 && priyoTrue
  priyoTell(total)
  priyoTell(ok)
}
```

## Comments

```priyo
// single-line comment

/*
multi-line comment
*/
```

## Assignment targets

```priyo
monalisa {
  priyoKeep profile = priyoCreate Person()
  profile.name = "Priyo"
}
```

<Callout title="Rule" type="error">
  The left side of `=` must be a valid assignable target like a variable, object field, or array
  index.
</Callout>
