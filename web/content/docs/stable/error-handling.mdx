# Error Handling

## Try / Catch / Finally

```js
prakritiTry {
  priyoTell(10 / 0)
} prakritiCatch (err) {
  priyoTell.Error(err.message)
} prakritiAtEnd {
  priyoTell.Info("done")
}
```

## Throw

```js
prakritiThrow "custom failure"
```

## Diagnostics

PriyoScript shows:

- Error code (example `PSYN-002`)
- Stage (`syntax`, `compile`, `runtime`)
- Source line and span caret (`^^^^`)
- Tips and docs links for faster fixes

Example:

```text
Code: PSYN-002
Stage: syntax
Location: file.priyo:3:10
Source: prakritiIff (x > 0) {
Span:          ^^^^^^^^^^
Docs: .../ERRORS.md#psyn-002
```

<Callout title="Title" type="error">
  Engine/internal errors indicate implementation issues, not user syntax mistakes.
</Callout>
