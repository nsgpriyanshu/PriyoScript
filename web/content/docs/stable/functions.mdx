# Functions

## Declare and call

```priyo
monalisa {
  lisaaTask add(a, b) {
    priyoGiveBack a + b
  }

  priyoTell(add(3, 4))
}
```

## Recursion

```priyo
monalisa {
  lisaaTask factorial(n) {
    prakritiIf (n <= 1) {
      priyoGiveBack 1
    } prakritiElse {
      priyoGiveBack n * factorial(n - 1)
    }
  }

  priyoTell(factorial(5))
}
```

## Closures (capturing outer scope)

```priyo
monalisa {
  lisaaTask makeCounter() {
    priyoChange count = 0

    lisaaTask next() {
      count = count + 1
      priyoGiveBack count
    }

    priyoGiveBack next
  }

  priyoKeep counter = makeCounter()
  priyoTell(counter())
  priyoTell(counter())
}
```

<Callout title="Function rule" type="warn">
  `priyoGiveBack` is valid only inside function bodies.
</Callout>
