# Functions

## Declare and Return

```js
lisaaTask add(a, b) {
  priyoGiveBack a + b
}

priyoTell(add(3, 4))
```

## Recursion

```js
lisaaTask factorial(n) {
  prakritiIf (n <= 1) {
    priyoGiveBack 1
  } prakritiElse {
    priyoGiveBack n * factorial(n - 1)
  }
}
```

## Closures

Functions can capture outer variables through lexical scope.

```js
lisaaTask makeCounter() {
  priyoChange count = 0
  lisaaTask next() {
    count = count + 1
    priyoGiveBack count
  }
  priyoGiveBack next
}
```

<Callout title="Title" type="warn">
  `priyoGiveBack` is valid only inside function bodies.
</Callout>
