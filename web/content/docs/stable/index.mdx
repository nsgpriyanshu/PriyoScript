# PriyoScript Stable Docs

PriyoScript is an interpreted language with readable custom keywords, a bytecode VM runtime, module support, OOP, arrays, and human-friendly diagnostics.

Use this section for stable features only.

## What You Can Learn Here

- How to install and run PriyoScript
- Core syntax and language constructs
- Modules/packages and project structure
- CLI, REPL, and error debugging
