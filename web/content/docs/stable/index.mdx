# PriyoScript Stable Docs

PriyoScript is an interpreted programming language with readable custom keywords, a bytecode VM, first-class arrays, modules/packages, OOP, and human-focused diagnostics.

This section documents **stable features only**.

<Callout title="Version Scope" type="warn">
  This section tracks stable behavior. Canary/experimental behavior may differ.
</Callout>

## First Program

```js title="hello.priyo"
monalisa {
  priyoTell("Hello, PriyoScript!")
}
```

Run it:

```bash
monalisa hello.priyo
```
