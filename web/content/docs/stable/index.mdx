# PriyoScript Handbook (Stable)

PriyoScript is a readable programming language with English-like keywords, a bytecode VM runtime, first-class arrays, modules, OOP, and humanized errors.

This documentation covers stable behavior only.

<Callout title="Version Scope" type="warn">
  Canary docs can change at any time. Use stable docs for real projects.
</Callout>

## First Program

```priyo title="hello.priyo"
monalisa {
  priyoTell("Hello, PriyoScript!")
}
```

Run it:

```bash
monalisa hello.priyo
```
